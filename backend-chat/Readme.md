# Backend-Chat

## Описание
Бэкенд для обработки логики чатов. Поддерживает:
- Создание и удаление чатрумов.
- Отправку и получение сообщений.
- Подключение через WebSocket для сообщений в реальном времени.

## Технологии
- Python (Django или Flask)
- PostgreSQL

## Установка
1. Создайте виртуальное окружение:
   ```bash
   python3 -m venv venv
   source venv/bin/activate```
2. Установите зависимости:
```bash
Копировать код
pip install -r requirements.txt```
3. Настройте переменные окружения в .env:
```python manage.py runserver```
