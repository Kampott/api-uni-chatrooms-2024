# University Chat Service

## Описание проекта
Сервис чатрумов для студентов и преподавателей университета. Позволяет создавать временные чатрумы для общения во время занятий, а также предоставляет удобный интерфейс для управления расписанием.

## Архитектура
Проект состоит из следующих модулей:
- **Backend-Chat**: Обрабатывает логику чатов.
- **Frontend-Chat**: Интерфейс для общения в чатах.
- **Backend-Site**: Управляет расписанием и создает чатрумы.
- **Frontend-Site**: Интерфейс для работы с расписанием и регистрации пользователей.

## Требования
- Python 3.9+
- Node.js 18+
- PostgreSQL
